写経ノート (Shakyo Note)
アプリの目的
毎日決まったテーマをノートに書き写すことで、「学びを習慣化」し、知識を定着させるためのツールです。
起動方法
1. このファイルを ⁠index.html⁠ として保存します。
2. GitHubリポジトリのルートに配置し、GitHub Pagesを有効にしてください。
使い方
1. テーマを選択して「ノートを生成」ボタンを押します。
2. 表示された内容を、手元の紙のノートに書き写してください。
3. 書き写したら、継続状況を確認しましょう。
